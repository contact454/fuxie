## 2. Quy tắc Từ vựng (Wortschatz)

### 2.1 Phạm vi Từ vựng

> [!IMPORTANT]
> Mọi từ vựng A1 PHẢI nằm trong hoặc dựa trên **Goethe A1 Wortliste** chính thức. Không đưa từ ngoài phạm vi.

- **Tổng mục tiêu:** 650–960 từ cốt lõi
- **Seed ban đầu:** Tối thiểu 500 từ (bao phủ đủ 14 chủ đề)
- **Mở rộng:** Tối đa 10% từ ngoài Wortliste nếu cần thiết cho ngữ cảnh
- **Loại trừ:** KHÔNG dùng từ thuộc B1+ (trừ khi là từ vay mượn quốc tế rất phổ biến)

### 2.4 Cấu trúc Dữ liệu Bắt buộc cho Mỗi Từ

> [!CAUTION]
> Mỗi từ vựng PHẢI có **ĐẦY ĐỦ** tất cả trường bắt buộc. Từ thiếu trường sẽ bị reject.

```typescript
interface VocabularyItem {
  // === BẮT BUỘC ===
  word: string              // Từ tiếng Đức (viết hoa danh từ)
  article?: 'der' | 'die' | 'das'  // Bắt buộc cho danh từ
  plural?: string           // Bắt buộc cho danh từ (die Bücher, die Äpfel, -)
  partOfSpeech: PartOfSpeech // Nomen | Verb | Adjektiv | Adverb | Präposition | Konjunktion | Artikel | Pronomen | Partikel
  meaningVi: string         // Nghĩa tiếng Việt
  meaningEn: string         // Nghĩa tiếng Anh
  cefrLevel: 'A1'           // Luôn là A1
  theme: A1Theme            // 1 trong 14 chủ đề
  
  // === BẮT BUỘC CHO DANH TỪ ===
  // article + plural phải chính xác 100%
  
  // === BẮT BUỘC CHO ĐỘNG TỪ ===
  conjugation?: {
    praesens: {              // Hiện tại
      ich: string
      du: string
      er_sie_es: string
      wir: string
      ihr: string
      sie_Sie: string
    }
    perfekt?: string         // hat/ist + Partizip II (nếu có trong A1)
    isIrregular: boolean     // Bất quy tắc?
    isSeparable?: boolean    // Trennbares Verb? (aufstehen → stehe auf)
    separablePrefix?: string // Prefix tách (auf-, an-, ein-, aus-...)
  }
  
  // === NÊN CÓ ===
  exampleSentences: string[] // Tối thiểu 2 câu, phù hợp A1
  audioUrl?: string         // URL audio phát âm
  ipa?: string              // Phiên âm IPA
  
  // === TÙY CHỌN ===
  notes?: string            // Ghi chú đặc biệt
  synonyms?: string[]       // Từ đồng nghĩa (trong A1)
  antonyms?: string[]       // Từ trái nghĩa (trong A1)
  collocations?: string[]   // Cụm từ phổ biến
  imageUrl?: string         // Hình minh họa
}
```

### 2.5 Quy tắc Chất lượng Từ vựng

| Quy tắc | Mô tả | Ví dụ đúng | Ví dụ sai |
|---------|-------|------------|-----------|
| **R1: Viết hoa** | Danh từ PHẢI viết hoa | der **T**isch | der **t**isch |
| **R2: Mạo từ** | Danh từ PHẢI kèm mạo từ | **der** Tisch | Tisch (thiếu mạo từ) |
| **R3: Số nhiều** | Danh từ PHẢI có dạng số nhiều | die Tische | (thiếu) |
| **R4: Chia động từ** | Động từ PHẢI có chia praesens đầy đủ | ich gehe, du gehst... | chỉ ghi "gehen" |
| **R5: Tách prefix** | Trennbare Verben PHẢI ghi rõ | **auf**stehen (trennbar) | aufstehen (không ghi chú) |
| **R6: Câu ví dụ** | Tối thiểu 2 câu, CHỈ dùng từ A1 | "Ich trinke Kaffee." | "Ich genieße Kaffee." (genießen = B1) |
| **R7: Nghĩa chính xác** | Nghĩa VN+EN phải chính xác, đầy đủ | fahren = lái xe, đi (phương tiện) | fahren = đi |
| **R8: Không bịa** | KHÔNG bịa mạo từ/số nhiều khi không chắc | (tra cứu trước) | der Mädchen (sai, = das) |

## 3. Quy tắc Ngữ pháp (Grammatik)

### 3.1 Danh sách Chủ đề Ngữ pháp A1 (Progression)

> [!IMPORTANT]
> Thứ tự dạy PHẢI tuân theo progression bên dưới. KHÔNG được nhảy cấp.

| Module | Chủ đề | Nội dung chi tiết | Ưu tiên |
|--------|-------|-------------------|---------|
| **G01** | Bảng chữ cái & phát âm | Das Alphabet, Aussprache, Umlaute (ä, ö, ü), ß | P0 |
| **G02** | Đại từ nhân xưng + sein/haben | ich, du, er/sie/es, wir, ihr, sie/Sie + chia sein/haben | P0 |
| **G03** | Chia động từ quy tắc (Präsens) | -e, -st, -t, -en, -t, -en | P0 |
| **G04** | Trật tự từ & câu hỏi | Verb Position 2, W-Fragen, Ja/Nein-Fragen | P0 |
| **G05** | Mạo từ xác định & bất định | der/die/das, ein/eine/ein (Nominativ) | P0 |
| **G06** | Phủ định | nicht, kein/keine/keinen | P0 |
| **G07** | Danh từ: giống, số nhiều, viết hoa | Genus, Plural, Großschreibung, Komposita | P0 |
| **G08** | Động từ bất quy tắc + đổi nguyên âm | sprechen→spricht, essen→isst, fahren→fährt | P1 |
| **G09** | Động từ khiếm khuyết (Modalverben) | können, müssen, möchten, wollen, dürfen, sollen | P1 |
| **G10** | Cách Akkusativ | den/einen/keinen (nam), Akkusativ-Verben | P1 |
| **G11** | Đại từ sở hữu | mein, dein, sein, ihr, unser, euer, ihr, Ihr | P1 |
| **G12** | Động từ tách (Trennbare Verben) | aufstehen, einkaufen, anfangen + Satzklammer | P1 |
| **G13** | Giới từ chỉ nơi chốn | in, an, auf, bei, nach, zu, von, aus (+ Dativ cơ bản) | P1 |
| **G14** | Mệnh lệnh thức (Imperativ) | du-Form, ihr-Form, Sie-Form | P1 |
| **G15** | Liên từ cơ bản | und, oder, aber, denn | P2 |
| **G16** | Giới từ chỉ thời gian | am, um, von...bis, im, seit | P2 |
| **G17** | Đại từ nhân xưng (Akkusativ) | mich, dich, ihn, sie, es, uns, euch, sie, Sie | P2 |
| **G18** | Perfekt cơ bản (giới thiệu) | haben/sein + Partizip II (chỉ giới thiệu, chưa chuyên sâu) | P2 |

### 3.2 Cấu trúc Bài Ngữ pháp

Mỗi bài ngữ pháp PHẢI có đủ 7 phần:

1. **Einführung** — Giới thiệu: Khi nào dùng? Tại sao quan trọng? (1-2 đoạn)
2. **Regeln** — Quy tắc: Công thức rõ ràng, có bảng biểu
3. **Beispiele** — Ví dụ: **Tối thiểu 5 câu**, tăng dần độ phức tạp, CHỈ dùng từ A1
4. **Ausnahmen** — Ngoại lệ: Các trường hợp bất quy tắc phổ biến
5. **Vergleich** — So sánh: So với tiếng Việt/Anh (giúp người học VN hiểu nhanh)
6. **Übungen** — Bài tập: **Tối thiểu 10 câu** luyện (có đáp án + giải thích)
7. **Eselsbrücke** — Mẹo ghi nhớ: Câu vần, hình ảnh, liên tưởng

### 3.3 Quy tắc Chất lượng Ngữ pháp

| Quy tắc | Mô tả |
|---------|-------|
| **G-R1** | Quy tắc PHẢI chính xác 100% — Không đơn giản hóa sai bản chất |
| **G-R2** | Ngoại lệ PHẢI được liệt kê rõ ràng, không giấu |
| **G-R3** | Ví dụ CHỈ dùng từ vựng trong phạm vi A1 Wortliste |
| **G-R4** | Bảng chia PHẢI đầy đủ tất cả ngôi (ich, du, er/sie/es, wir, ihr, sie/Sie) |
| **G-R5** | Bài tập PHẢI tăng dần độ khó (scaffolding) |
| **G-R6** | Phải có liên kết ngược (link đến bài ngữ pháp prerequisite) |
| **G-R7** | Thuật ngữ dùng SONG NGỮ: Akkusativ (cách 4 / accusative case) |

## 4. Quy tắc Thiết kế Bài tập (Übungen)

### 4.1 Dạng bài Bắt buộc cho A1

Mỗi dạng bài PHẢI mô phỏng chính xác format thi thực tế.

| Dạng bài (DE) | Dạng bài (VN) | Kỹ năng | Format thi | Thời gian |
|---------------|---------------|---------|------------|-----------|
| **Richtig/Falsch** | Đúng/Sai | Lesen, Hören | Goethe/Telc/ÖSD | 1-2 phút/câu |
| **Multiple Choice** | Trắc nghiệm 3 đáp án | Lesen, Hören | Goethe/Telc/ÖSD | 1-2 phút/câu |
| **Zuordnung** | Ghép đôi | Lesen | Goethe | 2-3 phút/câu |
| **Formular ausfüllen** | Điền biểu mẫu | Schreiben | Goethe/Telc | 5 phút |
| **Kurze Nachricht** | Viết tin nhắn ngắn (~30 từ) | Schreiben | Goethe/Telc/ÖSD | 15 phút |
| **Sich vorstellen** | Tự giới thiệu | Sprechen | Goethe/Telc/ÖSD | 3 phút |
| **Fragen stellen/beantworten** | Hỏi-đáp | Sprechen | Goethe/Telc/ÖSD | 5 phút |
| **Bitten formulieren** | Đưa yêu cầu | Sprechen | Goethe | 3 phút |
| **Lückentext** | Điền khoảng trống | Grammatik | Tự tạo | 1 phút/câu |
| **Wortschatzübung** | Bài tập từ vựng | Wortschatz | Tự tạo | 1 phút/câu |

### 4.2 Quy tắc Chất lượng Bài tập

| Quy tắc | Mô tả |
|---------|-------|
| **E-R1** | Mỗi bài tập PHẢI gắn: `cefr_level`, `skill`, `exam_type`, `estimated_time`, `grammar_topic` hoặc `theme` |
| **E-R2** | Đáp án PHẢI có **giải thích chi tiết** (tại sao đúng/sai) |
| **E-R3** | Mỗi đáp án sai PHẢI có giải thích **tại sao sai** (distractor analysis) |
| **E-R4** | Ngữ liệu (đoạn đọc, audio script) CHỈ dùng từ A1 (±10% A2 để thách thức) |
| **E-R5** | Độ khó trong cùng bài PHẢI tăng dần (câu 1 dễ → câu cuối khó nhất) |
| **E-R6** | Bài nghe PHẢI có transcript |
| **E-R7** | Bài viết PHẢI có rubric chấm điểm theo format thi target |

### 4.3 Rubric Chấm Bài Viết A1 (theo Goethe)

| Tiêu chí | 5 điểm | 3 điểm | 0 điểm |
|----------|--------|--------|--------|
| **Inhalt** (Nội dung) | Đủ 3 điểm yêu cầu | 2/3 điểm | 0-1 điểm |
| **Kommunikative Gestaltung** | Mạch lạc, giao tiếp hiệu quả | Có lỗi nhưng hiểu được | Không hiểu được |
| **Korrektheit** | Ít lỗi, không ảnh hưởng hiểu | Có lỗi nhưng chấp nhận được | Nhiều lỗi, khó hiểu |

## 5. Quy tắc Nội dung Đọc/Nghe (Lesen & Hören)

### 5.1 Quy tắc Văn bản Đọc A1

| Quy tắc | Mô tả |
|---------|-------|
| **L-R1** | Chỉ dùng câu **ngắn, đơn giản** (tối đa 10-12 từ/câu) |
| **L-R2** | Loại văn bản: biển báo, email ngắn, quảng cáo, ghi chú, biểu mẫu |
| **L-R3** | Độ dài tối đa: 50-80 từ/đoạn đọc |
| **L-R4** | Từ vựng: 95% từ A1 Wortliste, 5% đoán được từ ngữ cảnh |
| **L-R5** | KHÔNG có mệnh đề phụ phức tạp (weil, obwohl, damit...) |
| **L-R6** | Font rõ ràng, cỡ chữ đủ lớn |

### 5.2 Quy tắc Audio A1

| Quy tắc | Mô tả |
|---------|-------|
| **H-R1** | Tốc độ nói: **CHẬM và RÕ RÀNG** (~100-120 từ/phút) |
| **H-R2** | Hochdeutsch (tiếng Đức chuẩn), KHÔNG phương ngữ |
| **H-R3** | Google TTS voice: `de-DE-Wavenet-A/B/C/F` |
| **H-R4** | Pause rõ ràng giữa các câu (1-2 giây) |
| **H-R5** | Nghe tối đa 2 lần cho mỗi bài |
| **H-R6** | Loại audio: hội thoại ngắn, tin nhắn điện thoại, thông báo công cộng |
| **H-R7** | Mỗi audio phải có **transcript** kèm theo |

## 7. Checklist Kiểm soát Chất lượng

### 7.1 Checklist cho Từ vựng

- [ ] Từ nằm trong Goethe A1 Wortliste?
- [ ] Danh từ: có mạo từ đúng (der/die/das)?
- [ ] Danh từ: có dạng số nhiều?
- [ ] Động từ: có chia praesens đầy đủ 6 ngôi?
- [ ] Động từ: đánh dấu bất quy tắc/tách prefix?
- [ ] Nghĩa tiếng Việt + Anh chính xác?
- [ ] Có tối thiểu 2 câu ví dụ (chỉ dùng từ A1)?
- [ ] Gắn đúng 1 trong 14 chủ đề?
- [ ] Không trùng lặp với từ đã có?

### 7.2 Checklist cho Ngữ pháp

- [ ] Thuộc danh sách G01-G18?
- [ ] Có đủ 7 phần (Einführung → Eselsbrücke)?
- [ ] Quy tắc chính xác 100%?
- [ ] Tối thiểu 5 ví dụ (tăng dần độ khó)?
- [ ] Tối thiểu 10 bài tập (có đáp án + giải thích)?
- [ ] Ví dụ chỉ dùng từ vựng A1?
- [ ] Có so sánh với tiếng Việt/Anh?

### 7.3 Checklist cho Bài tập

- [ ] Có metadata đầy đủ (cefr_level, skill, exam_type...)?
- [ ] Đáp án có giải thích chi tiết?
- [ ] Distractor analysis cho đáp án sai?
- [ ] Ngữ liệu chỉ dùng từ A1?

### 7.4 Checklist cho Nội dung Đọc

- [ ] Tối đa 10-12 từ/câu?
- [ ] Tối đa 80 từ/đoạn?
- [ ] 95% từ A1?
- [ ] Đúng loại văn bản A1 (email, biển báo, quảng cáo...)?

### 7.5 Checklist cho Audio

- [ ] Hochdeutsch, không phương ngữ?
- [ ] Tốc độ chậm, rõ ràng?
- [ ] Có transcript?
- [ ] Pause giữa các câu?

## 8. Danh sách Đen (Blacklist)

> [!WARNING]
> Các yếu tố sau KHÔNG ĐƯỢC xuất hiện trong nội dung A1:

### Ngữ pháp cấm ở A1
- ❌ Konjunktiv I, II
- ❌ Passiv (câu bị động)
- ❌ Plusquamperfekt
- ❌ Futur I, II
- ❌ Relativsätze (mệnh đề quan hệ)
- ❌ Indirekte Rede (lời nói gián tiếp)
- ❌ Partizipialkonstruktionen
- ❌ Liên từ phụ: obwohl, nachdem, bevor, während, damit
- ❌ Genitiv (cách 2)

### Từ vựng cấm ở A1
- ❌ Từ trừu tượng (Demokratie, Philosophie, Umweltschutz...)
- ❌ Thuật ngữ chuyên ngành
- ❌ Thành ngữ phức tạp
- ❌ Từ viết tắt không phổ biến
- ❌ Từ lóng (Umgangssprache) — trừ "Na?" "Tschüss"
